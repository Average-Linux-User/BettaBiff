Use the ask biff command to open up a custom AI agent as long as you have Ollama installed. Also Eat exits the program, explorer open file explorer, and the rest are easter eggs for you to find out for more info type help into the program. Also for the AI part to work you must have Ollama installed and have navigated to the directory where the program lives and run the command "ollama create biff -f ./Modelfile" If you are trying to modify the code here is a quick tip, biff.py if the script for the AI part that runs when you type "ask biff" in the main assistant.py file. assistant.py is the main part of the program biff.py is the helper.
